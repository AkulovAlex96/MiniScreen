# MiniScreen — демо виджетов и анимаций

8 виджетов на круглом GC9A01 240x240. Подключение то же, что в
`../firmware_demo/WIRING.md`.

## Виджеты (BTN1 переключает по кругу)

| # | Виджет | Что показывает |
|---|--------|----------------|
| 0 | Analog clock | Циферблат с рисками, плавная секундная стрелка, дата |
| 1 | Digital clock | HH:MM 7-сегментным шрифтом, мигающее двоеточие, секундная дуга по краю |
| 2 | Calendar | Сетка месяца, сегодняшний день подсвечен, выходные красным |
| 3 | Ring loader | Круговой прогресс с градиентом красный→зелёный и точкой-головой |
| 4 | Progress bar | Плоский бар с бегущим бликом и анимированными точками |
| 5 | Gauge | Стрелочный индикатор со шкалой 0–100, «дрейфующее» значение |
| 6 | Matrix | Зелёный дождь символов с хвостами разной скорости |
| 7 | Starfield | Полёт сквозь звёзды (3D-проекция) |

BTN2 — вкл/выкл подсветку.

## Архитектура

- Рендер в **полнокадровый спрайт** (240x240x16bit = 115 КБ в heap),
  вывод одним `pushSprite` — без мерцания
- SPI 40 МГц (если артефакты на брэдборде — снизить `freq_write` до 20 МГц)
- Часы стартуют от времени компиляции (`__TIME__`/`__DATE__`) + `millis()`.
