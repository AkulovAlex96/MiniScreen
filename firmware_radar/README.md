# MiniScreen — радар самолётов

Круговой радар вокруг заданных координат: ESP32-S3 опрашивает OpenSky Network
по Wi-Fi (без ключа, анонимный доступ) и рисует борта в воздухе как блипы —
дистанция от центра и азимут пересчитаны из lat/lon в декартовы координаты.

## Запуск

Скопировать [src/secrets.h.example](src/secrets.h.example) в `src/secrets.h` и заполнить:

```cpp
#define WIFI_SSID_1 "приоритетная_сеть"
#define WIFI_PASS_1 "пароль"
#define WIFI_SSID_2 "резервная_сеть"     // например, точка доступа телефона
#define WIFI_PASS_2 "пароль"

#define HOME_LAT 55.7558   // домашние координаты — центр радара
#define HOME_LON 37.6173
```

(`secrets.h` в .gitignore, в репозиторий не попадает). Ключ API не нужен —
OpenSky отдаёт `states/all` анонимно. Прошить: `pio run --target upload`.

## Управление

- **BTN1** — рефреш данных прямо сейчас (не ждать таймер)
- **BTN2** — подсветка

## Особенности

- Радиус отображения — 50 км (`RANGE_KM` в `main.cpp`), опрос раз в 60 сек
  (`POLL_INTERVAL_MS`) — анонимный доступ OpenSky ограничен ~400 "кредитами"/день,
  не увеличивайте частоту опроса сильно
- Самолёты на земле (`on_ground=true`) не показываются
- JSON парсится через `DeserializationOption::Filter` — держим в памяти только
  нужные поля (callsign/координаты/высоту/скорость/курс), не весь ответ
- Внизу экрана — данные ближайшего борта (позывной, высота, скорость)
