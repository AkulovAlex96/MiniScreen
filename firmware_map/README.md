# MiniScreen — карта города + погода текстом

Статичная карта вокруг заданных координат (Mapbox Static Images API) с
плашкой погоды поверх (Open-Meteo, без ключа). Карта фетчится редко (при
старте и по BTN1) — не тратим лимиты и не гоняем JPEG по сети каждую минуту.

## Запуск

Скопировать [src/secrets.h.example](src/secrets.h.example) в `src/secrets.h` и заполнить:

```cpp
#define WIFI_SSID_1 "приоритетная_сеть"
#define WIFI_PASS_1 "пароль"
#define WIFI_SSID_2 "резервная_сеть"     // например, точка доступа телефона
#define WIFI_PASS_2 "пароль"

#define HOME_LAT 55.7558   // центр карты
#define HOME_LON 37.6173

#define MAPBOX_TOKEN "..."   // см. ниже
```

(`secrets.h` в .gitignore). Прошить: `pio run --target upload`.

### Нужен ключ: Mapbox access token

1. Зарегистрироваться на [mapbox.com](https://www.mapbox.com) (бесплатно)
2. Взять Default public token (или создать новый) в аккаунте
3. Вписать в `MAPBOX_TOKEN`

Free tier — 50 000 запросов/мес к Static Images API, нам хватит с огромным
запасом (карта фетчится редко).

## Управление

- **BTN1** — рефреш карты и погоды прямо сейчас
- **BTN2** — подсветка

## Особенности

- Карта запрашивается сразу в 240x240 (без `@2x`, без тайловой математики —
  Mapbox сам отдаёт готовое изображение по центру/зуму), декодируется
  JPEGDEC — тот же паттерн, что в `firmware_mjpeg`
- `MAP_ZOOM` (по умолчанию 14) — масштаб карты, крутится в `main.cpp`
- Погода обновляется раз в 5 минут (`WEATHER_INTERVAL_MS`) и рисуется
  плашкой поверх уже загруженной карты — карта не перезапрашивается
- Рассматривался вариант без ключа (библиотека `OpenStreetMap-esp32`,
  тянет тайлы OSM без токена), но она требует реально работающий PSRAM,
  а плата в `platformio.ini` (`esp32-s3-devkitc-1`) — вариант без PSRAM.
  Mapbox+JPEGDEC гарантированно работает на той же конфигурации, что и
  остальные прошивки в репозитории
