# MiniScreen — стилизованные тучи по реальным данным

Процедурные "тучи" на экране, но не декоративные — их количество, темнота
неба и дождь управляются реальными данными погоды (Open-Meteo, без ключа).
Дрейф туч по экрану — по направлению и скорости ветра.

## Запуск

Скопировать [src/secrets.h.example](src/secrets.h.example) в `src/secrets.h` и заполнить:

```cpp
#define WIFI_SSID_1 "приоритетная_сеть"
#define WIFI_PASS_1 "пароль"
#define WIFI_SSID_2 "резервная_сеть"     // например, точка доступа телефона
#define WIFI_PASS_2 "пароль"

#define HOME_LAT 55.7558   // для какой точки брать погоду
#define HOME_LON 37.6173
```

(`secrets.h` в .gitignore). Ключ API не нужен — Open-Meteo открытый. Прошить:
`pio run --target upload`.

## Управление

- **BTN1** — рефреш погоды прямо сейчас
- **BTN2** — подсветка

## Особенности

- Опрос раз в 5 минут (`POLL_INTERVAL_MS`), сама анимация (дрейф/дождь) —
  каждый кадр независимо от опроса сети
- Кол-во видимых туч ~ `cloud_cover` (0-100%), темнота неба и туч растёт с
  облачностью, частицы дождя появляются при `precipitation > 0`
- Направление дрейфа — куда дует ветер (`wind_direction_10m` + 180°, так как
  метео-направление — это "откуда"), скорость — от `wind_speed_10m`
- Температура крупным текстом сверху, `cloud_cover`% — снизу
